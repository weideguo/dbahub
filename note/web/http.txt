# 可以通过telnet模拟tcp层发起http请求，两个回车则提交
telnet 10.0.0.1 80

## GET请求
GET / HTTP/1.1                      #进入telnet命令行 构造GET请求 / 为路径
User-Agent: curl/7.29.0             #
Accept: */*                         #

##

## POST请求
POST / HTTP/1.1
User-Agent: curl/7.29.0
Content-Length: 8
Content-Type: application/x-www-form-urlencoded

abcdefgh

###


HTTP方法：
GET     从服务器上获取一个资源 
POST    向服务器提交数据（可能创建一个新的资源）
PUT     向服务器提交数据
DELETE  删除指定数据
HEAD    只请求网页的头部信息
状态码：
200     正常，请求成功
301     永久移动，一般用于域名重定向
304     未修改，一般用于缓存
401     禁止访问，未授权
403     禁止访问，通常代表认证通过，但没有访问权限
404     未找到资源
500     服务器内部错误
