LDAP Lightweight Directory Access Protocol 
CS模式
LDAP树形数据库如下：
dn ：一条记录的详细位置
dc ：一条记录所属区域      
ou ：一条记录所属组织      
cn/uid：一条记录的名字/ID  
dn:cn=L20,ou=L13,ou=L12,ou=L11,dc=L0


权限树
 L0                              #dc
 ├── L11                         #ou
     ├── L12                     #ou
         ├── L13                 #ou
             ├── L20             #cn
             ├── L21             #cn
             ├── L22             #
             ├── L23             #
             ├── L24             #
             ├── L25             #
             └── L26             #



Pluggable Authentication Modules
Linux-PAM + OpenLDAP 实现统一的linux账号管理

yum install openldap openldap-clients openldap-servers


# 登录并获取详细信息
ldapsearch -x -H ldap://ldap.my_domain.abc:389    \
  -D "uid=my_account,ou=people,dc=a,dc=com"       \
  -b "dc=a,dc=com"                                \
  "(uid=my_account)" -w 'my_password'


# 简单登录验证
ldapwhoami  -x -H ldap://ldap.my_domain.abc:389   \
  -D "uid=my_account,ou=people,dc=a,dc=com"       \
  -w 'aaabbb123'
